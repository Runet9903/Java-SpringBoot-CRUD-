# README

## Project 紹介
JavaとSpring bootを利用してCRUD機能を持った掲示板を簡単に作ってみました。

## 開発期間
- 2023.12.15 ~ 2024.02.18

## 主要機能
- 掲示文を作成し、確認が可能で、修正と削除が可能です。

## 開発環境
- Version : Java 17
- IDE : IntelliJ
- Framework : SpringBoot 3.2.2
